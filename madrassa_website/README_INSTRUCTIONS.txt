📁 فائلز کی فہرست:
  1_HOME_PAGE.html      — ہوم پیج
  2_ABOUT_PAGE.html     — ہمارے بارے میں
  3_COURSES_PAGE.html   — کورسز / شعبہ جات
  4_ADMISSION_PAGE.html — داخلہ
  5_CONTACT_PAGE.html   — رابطہ
  ALL_PAGES_COMBINED.html — سب ایک جگہ

📋 WordPress میں لگانے کا طریقہ:
  1. WordPress Dashboard میں لاگ اِن کریں
  2. Pages → Add New پر جائیں
  3. پیج کا نام لکھیں (مثلاً "Home")
  4. ایڈیٹر میں اوپر دائیں طرف ⋮ (تین نقطے) پر کلک کریں
  5. "Code Editor" یا "Edit as HTML" منتخب کریں
  6. متعلقہ فائل کا HTML کوڈ کاپی کر کے پیسٹ کریں
  7. "Publish" پر کلک کریں
  8. باقی تمام پیجز کے لیے یہی عمل دہرائیں

⚙️ ضروری سیٹنگز:
  - Settings → Reading → Homepage = Static Page → "Home" سلیکٹ کریں
  - Appearance → Menus → نیا مینیو بنائیں → سب پیجز ایڈ کریں → Primary Menu سیٹ کریں
